Project voor AI in de IoT minor

Einddoel: home assistant genaamd 'homie' die spraak kan herkennen
commando's:
- "Hey Homie" (wake up commando)
- "What is the temperature"
- "lights off"
- "Lights on"
bij deze commando's kunnen ook acties uitgevoerd worden. 
Er kan bij "what's the temperature" ook een LCD schermpje gebruikt worden die de temperatuur weergeeft.
